Ola pessoal! Estou com algumas duvidas sobre o modulo de Excel Avancado, especificamente na parte de Tabelas Dinamicas.

Quando tento criar uma tabela dinamica a partir de uma base de dados grande (mais de 50.000 linhas), o Excel fica muito lento e as vezes trava. Alguem sabe se existe alguma configuracao ou tecnica para otimizar esse processo?

Alem disso, gostaria de entender melhor como usar os campos calculados dentro das tabelas dinamicas. No video da aula (28:34) o instrutor menciona sobre isso rapidamente, mas nao consegui aplicar na pratica.

Agradeco qualquer ajuda!

Ola Carlos! Sobre a lentidao com bases grandes, existem algumas tecnicas que podem ajudar:

1. Use o Power Query para limpar e transformar os dados antes de criar a tabela dinamica. Isso reduz significativamente o processamento.

2. Desative a atualizacao automatica da tabela e atualize manualmente quando necessario.

3. Se possivel, trabalhe com um subconjunto dos dados para analises exploratorias.

Complementando a resposta da Ana: para campos calculados, voce precisa ir em Analise de Tabela Dinamica > Campos, Itens e Conjuntos > Campo Calculado.

La voce pode criar formulas personalizadas usando os campos existentes. Por exemplo, se voce tem "Vendas" e "Custos", pode criar um campo "Margem" com a formula =Vendas-Custos.

Fernando, para sincronizar slicers com multiplas tabelas dinamicas, voce precisa clicar com o botao direito no slicer, ir em "Conexoes de Relatorio" e marcar todas as tabelas dinamicas que deseja controlar.

Lembre-se que todas as tabelas precisam estar baseadas na mesma fonte de dados para isso funcionar corretamente!
